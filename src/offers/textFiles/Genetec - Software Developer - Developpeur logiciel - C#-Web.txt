The team:

Join a passionate team who is working with the latest technologies in software development.

Here is an overview of the stack used in the context of the project:

A microservice ecosystem using the latest C#- .NET Core running on Linux
Diversified and modern data layer using NoSQL databases like MongoDB, Elasticsearch and Redis
Asynchronous and eventually consistent approach using an event-based system (AMQP) to propagate changes
Container-based deployments using Docker and Kubernetes
Infrastructure as code using Terraform and Argo CD
Quality assurance with great coverage of unit, functional and integration tests
Continuous integration and continuous delivery (CI-CD) using Azure DevOps
Feature flagging, A-B testing, Canary release using Launch Darkly
