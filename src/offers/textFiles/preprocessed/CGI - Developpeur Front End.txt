Notre client du secteur municipal a entamé un grand chantier visant à réviser ses règlements d'urbanisme et a saisi cette occasion pour examiner en profondeur sa réglementation dans le cadre d'une refonte complète. Afin de couvrir les besoins de développement de la phase de construction du projet, le service de l'Innovation et des Technologies est actuellement à la recherche d'un Intégrateur pour assurer l'intégration, la coordination et le lead technique au sein du projet. La ressource identifiée collaborera étroitement avec le chargé de projet, l'analyste d'affaires et les autres Développeurs. Elle devra également se coordonner avec la personne responsable de l'architecture TI.

Fonctions et responsabilités

Dans le cadre de ce projet, la méthodologie Agile sera utilisée dans la mesure du possible. Le Développeur assurera le développement des livrables qui lui seront confiés. Le livrable principal sera un site Web responsive. Incluant des fonctions de recherche tant dans des éléments de textes, que des éléments cartographiques ou mixtes. Il est possible que le Développeur ait des API à développer ou utiliser pour connecter divers systèmes liés au projet.

Maîtrise du développement Back end
Maîtrise du JavaScript, React, Gatsby (requis obligatoire!)
Expérience dans la réalisation d'interface HTML5, CSS et JavaScript avec des cartes web (géospatial).
Expérience en React JS
Expérience TDD (Test Driven Development)
Expérience pratique DevSecOps
Expérience avec le Web mapping (bonne connaissance d'une librairie : MapBox, Leaflet, ESRI JsAPI, ou Google)
Expérience avec les services REST sécurisés par OpenID Connect
Expérience avec le développement d'API en JavaScript
Expérience avec SQL Server
Expérience dans la gestion d'équipe de Développeurs

Gestionnaire de code source GIT, TFS

Compétences - connaissances particulières requises

Expérience de 5 ans minimum dans le domaine
